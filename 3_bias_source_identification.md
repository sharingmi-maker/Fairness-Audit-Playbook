# Bias Source Identification

## Purpose
To identify technical and organizational mechanisms that cause unfair outcomes.

## Common Bias Sources
- Data representation bias
- Labeling bias
- Optimization bias
- Feedback loops
- Deployment and human override bias

## Required Actions
- Conduct causal and counterfactual analysis
- Trace disparities back to their sources

## Intersectional Fairness
Evaluate how bias sources disproportionately affect intersectional groups.

## Output
A bias source mapping document.
